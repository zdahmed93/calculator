# Simple Calculator

A minimal calculator application for testing code review tools.

## Structure

- `index.js` - Main application entry point
- `calculator.js` - Calculator class with utility functions (all in one file)

## Usage

```bash
node index.js
```

## Known Issues (for testing)

This project intentionally contains issues to test code review:
- Division by zero not handled
- Unused function `unusedFunction`
- Missing input validation
